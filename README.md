# Python_Typing_Speed_Test
The Typing Speed Test is a Python project that tests your typing speed. 
This project is a command-line application that prompts the user to type a random passage of text, records their keystrokes, and calculates their typing speed in WPM(Words per minute).
This project works best in the terminal.
In this project you have right words typed is shown in green and wrong words are shown in red.
The timer stops when all the words have been typed correctly i.e. all charaters are green.
